# Vue 3 Starter

A basic bare bones starter for Vue projects

#### What it uses:

- Vite & Vue
- Tailwind CSS
- eslint
- yarn

#### To get started:

After cloning the repo, do the following

```shell
# install dependencies
$ yarn
# run it locally at localhost:3000
$ yarn dev
```

Fix any ESLint warnings with `yarn lint:fix`.
